# Quick Start Guide

## Setup (One-time)

```bash
cd SafeCall
npm install
```

## Run Development Server

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.

## What to Show in Your Presentation

1. **Landing Page**: Scroll through all sections to show the complete story
2. **Demo Page**: 
   - Click one of the example scripts (e.g., "Bank Scam")
   - Click "Analyze Call" to see the risk analysis in action
   - Try typing your own text to show interactivity
3. **Pricing Page**: Show the transparent pricing model

## Key Features to Highlight

- ✅ Senior-friendly design (large text, high contrast)
- ✅ Interactive demo that actually works
- ✅ Clear value proposition
